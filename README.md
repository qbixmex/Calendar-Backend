# Calendar Backend

This is the backend for "React Calendar App" using "NodeJs", "ExpressJs" and "MongoDB".

## Install Dependencies

NPM
```
npm install
```

YARN
```
yarn
```

## Run Production

NPM
```
npm start
```

YARN
```
yarn start
```

## Run Development

NPM
```
npm run dev
```

YARN
```
yarn dev
```
